Hide Deprecated Applications=Cacher les applications obsolètes 
This setting hides deprecated applications from the list of available apps Deprecated apps will still however display within the installed and previous apps sections=Ce paramètre cache les applications obsolètes de la liste des applis disponibles. Cependant les applis obsolètes apparaîtront toujours dans les sections "Applis Installées" et "Applis Précédentes".
Hide Incompatible Applications=Cacher les applications incompatibles
This setting will hide applications that are listed as being incompatible with your version of Unraid=Ce paramètre cache les applications listées commme incompatibles avec votre version de Unraid
Enable Reinstall Default=Activer Réinstallation par défaut
This option enables another button on the cards and popups which will allow you to reinstall an installed application using its default template values This option can be confusing at times, and is only generally used if you wish to install and have an application running twice simultaneously=Un nouveau bouton sera disponible dans les vignettes et fenêtres pop-up permettant l'installation supplémentaire, avec valeurs par défaut, d'une application déjà installée. Cette option peut parfois prêter à confusion, et est normalement utilisée lorsque vous désirez exécuter deux fois la même application simultanément
Enable additional search results from dockerHub=Ajouter les résultats de recherche provenant de DockerHub
This setting enables CA to retrieve additional search results from dockerHub Note that docker must be enabled for this to be operational=Ce paramètre permet d'ajouter les résultats provenant de DockerHub dans la recherche de CA. Notez que docker doit être activé pour que ce dernier soit opérationnel
Allow CA to send any emergency notifications=Autoriser CA à envoyer des notifications d'urgence
This setting allows CA to periodically scan your system and alert you to the presence of=Ce paramètre permet à CA d'effectuer des analyses périodiques de votre système et de vous aviser lors de la présence:
1 Known malware and security vulnerabilities within applications=1 De logiciels malveillants et failles de sécurité connues dans les applications
2 Critical security vulnerabilities within Unraid OS=2 De Failles de sécurité critiques au sein du système d'exploitation Unraid
3 Any installed applications which may seriously hinder the operation of your server=3 D'applications installées pouvant gravement entraver le bon fonctionnement de votre serveur
